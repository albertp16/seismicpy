## 📖 Overview

SeismicPy is an open-source Python library for **seismic design and analysis**, leveraging **NSCP 2015** and industry standards. The library aims to provide engineers with a reliable toolkit for base shear calculations, response spectrum generation, and other seismic-related computations.
